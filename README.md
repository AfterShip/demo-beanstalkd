# demo-beanstalkd

Demo BS

## Requirement
- Redis
- Beanstalkd

## How to run
Clone project first
```
git clone git@github.com:AfterShip/demo-beanstalkd.git

```

run server
```

cd server && npm install && npm start

```

run worker
```

cd worker && npm install && npm start

```
